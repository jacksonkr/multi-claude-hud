Multi-Claude HUD — Settings

About & what the lights mean
The overlay shows a light per Claude Code terminal. It scans this machine (and any running WSL distros) every few seconds; turn on LAN sharing below — with the same Network key on each device — to see terminals across your machines. A hub is optional (for cross-subnet aggregation and history).
Working — actively thinking or running a tool (incl. a monitor)
Waiting on you — needs an answer (a prompt or permission)
Done + subprocess — Claude finished (red), but a shell/monitor is still working (green)
Idle — finished, nothing pending
The number over a light shows how long it's been in that state (waiting / idle) or how long the subprocess has been running (split). Favorites (★) pin to the top. Hover the panel to bring it forward; click a light to see behind it; click the “CLAUDE” header to collapse.
